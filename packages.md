# Debian 13 Package List

## 1️⃣ Base System / Window Management
- **tuigreet** – Greeter/login manager for Sway.  
- **sway** – Tiling Wayland compositor.  
- **swaybg** – Background setting utility for Sway.  
- **swayidle** – Idle management daemon for Sway.  
- **swaylock** – Screen locker for Sway.  
- **swayosd** – On-screen display for Sway notifications.  
- **sway-notification-center** – Notification system for Sway.  
- **sway-backgrounds** – Collection of wallpapers for Sway.  
- **wofi** – Application launcher/menu for Wayland.  
- **waybar** – Highly configurable status bar for Sway.  
- **xwayland** – X server compatibility layer for Wayland.  

## 2️⃣ Terminal & CLI Tools
- **kitty** – GPU-based terminal emulator.  
- **vim** – Classic text editor.  
- **neovim** – Modern Vim fork with extra features. *(built from source)*  
- **tmux** – Terminal multiplexer.  
- **wl-clipboard** – Clipboard manager for Wayland.  
- **bat** – Enhanced `cat` with syntax highlighting.  
- **fzf** – Fuzzy file finder.  
- **ripgrep** – Fast text search in files.  
- **fd-find** – Simple and fast alternative to `find`.  
- **eza** – Modern replacement for `ls` with colors and icons.  
- **zip / unzip** – File archiving tools.  
- **zoxide** – Smart directory jumper.  
- **procs** – Modern process viewer.  
- **duf** – Disk usage utility.  
- **brightnessctl** – Adjust screen brightness.
- **nalaz** – A nala wrapper with fzf for package search with integrated preview.  

## 3️⃣ Version Control / Development
- **git** – Version control system.  
- **lazygit** – Terminal UI for Git.  
- **lazydocker** – Terminal UI for Docker. *(built from source)*  
- **stow** – Symlink manager for dotfiles.  

## 4️⃣ System Monitoring & Utilities
- **btop** – Terminal-based system monitor.  
- **neofetch** – Display system information in terminal.  
- **lm-sensors** – Hardware monitoring for temperatures and fans.  
- **cava** – Terminal audio visualizer.  
- **playerctl** – Control media players from CLI.  
- **grimshot** – Screenshot tool for Wayland.  
- **cbonsai** – Wayland image/graphics utility.  

## 5️⃣ Audio / Media
- **pipewire-audio** – Sound server for modern Linux setups.  
- **mpv** – Video and audio player.  


## 6️⃣ Applications
- **firefox-esr** – Web browser.  
- **thunderbird** – Email client.  
- **durdraw** – Drawing application.  
- **gimp** – Image editor.  
- **virt-manager** – Virtual machine manager.  
- **termshark** – Terminal-based network analyzer.  
- **vscode** – Code editor. *(installed via .deb)*  
- **anytype** – Personal knowledge management app. *(installed via .deb)*  
- **jetbrains-toolbox** – JetBrains IDE manager. *(built from source)*  
- **yazi** – Text or development tool. *(built from source)*
